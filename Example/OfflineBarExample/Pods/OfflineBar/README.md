## Requirements ##
* Swift 3.0
* iOS 8.0+
* Xcode 8

## Installation ##
#### Cocoapods ####
- Add into your Podfile.

```:Podfile
pod "OfflineBar"
```

Then `$ pod install`
- Add `import OfflineBar` to the top of your files where you wish to use it.
